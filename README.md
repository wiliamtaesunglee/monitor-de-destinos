
<!-- USAGE -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/wiliamtaesunglee/monitor-de-destinos.git
   ```
2. Install NPM package
   ```sh
   npm install
   ```

### Running local

   ```sh
   npm run start
   ```

### Build

   ```sh
   npm run build
   ```
